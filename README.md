[FlapPyBird-RL]
===============

A Reinforcement Learning played Flappy Bird. It's forked from [FlaPyBird](https://github.com/sourabhv/FlapPyBird) which is based on [python-pygame](https://github.com/pygame/pygame).

## How To Run

Please use `uv` to run this project.

```
uv run main.py
```
